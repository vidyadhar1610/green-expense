# Green Expense

Green Expense is a personal finance management application that helps users track expenses, manage debts, and monitor savings.

## Tech Stack

* React
* Node.js
* Express
* MongoDB

## Features

* Add and track expenses
* Manage given/taken debts
* Savings tracking dashboard
* Authentication system

## Installation

1. Clone the repo
2. Run `npm install`
3. Run `npm start`
